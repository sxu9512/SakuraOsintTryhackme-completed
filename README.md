## 🧭 Investigation areas covered

### 🧩 SVG metadata extraction

The case uses SVG files as a clue source. You can inspect file details, hidden text, and metadata that may point to usernames, tool names, or saved paths.

### 👤 Identity resolution

The writeup shows how to connect the same person across different sites. It uses usernames, profile photos, writing style, and shared details to match accounts.

### 💰 Cryptocurrency tracing

You can follow wallet activity, address reuse, and transfers that help link public names to blockchain records.

### ⛓️ Blockchain forensics

The repository includes the kind of checks used on blockchain explorers such as transaction history, token movement, and wallet relationships.

### 📱 Social media profiling

The investigation uses public posts, profile changes, and account links to build a timeline.

### 📡 WiFi geolocation

The case also uses WiFi and network clues to narrow down places tied to the subject’s movement.

### 🗺️ Travel route reconstruction

Public data points are combined to rebuild travel paths and time order, which helps confirm location claims.
